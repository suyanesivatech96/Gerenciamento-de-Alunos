# Gerenciamento-de-Alunos
Aplicação em Python com interface simples para cadastrar, listar e organizar informações de alunos. O sistema utiliza banco de dados relacional para persistência de dados e operações CRUD (Create, Read, Update, Delete). Foram aplicadas consultas SQL para gerenciamento das informações e extração de dados.

Tecnologias usadas:  Python e SQL (MySQL)

Objetivo do projeto:
Demonstrar lógica de programação, integração com banco de dados e capacidade de estruturar e manipular dados de forma eficiente.
